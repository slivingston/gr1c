◆ get_offsets()

int* get_offsets ( char *  metric_vars,
int *  num_vars 
)

Return an array of length 2n, where n is the number of entries in metric_vars. The values at indices 2i and 2i+1 are the offset and width of the i-th variable's binary representation. Return NULL if error. metric_vars is a space-separated list of variables to use in computing distance. The caller is expected to free the array.

This function assumes that evar_list and svar_list have not been linked. get_offsets_list() is a more general version.
